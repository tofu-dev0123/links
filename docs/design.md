## UI設計

### レイアウト

| 画面サイズ | レイアウト   |
| ---------- | ------------ |
| スマホ     | 縦並びカード |
| PC         | 2列グリッド  |

### デザイン方針

| 項目           | 方針                       |
| -------------- | -------------------------- |
| ダークモード   | 対応しない                 |
| カードアイコン | 各サービスのアイコンを表示 |
| レスポンシブ   | スマホ優先で設計           |

## データ設計

### データ管理

リンクデータは `src/data/links.ts` で管理する。

### リンクのデータ構造

| フィールド    | 型        | 内容         |
| ------------- | --------- | ------------ |
| `name`        | `string`  | サービス名   |
| `url`         | `string`  | リンク先URL  |
| `icon`        | `string`  | アイコン     |
| `description` | `string`  | 簡単な説明文 |
| `visible`     | `boolean` | 表示フラグ   |
